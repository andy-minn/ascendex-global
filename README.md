# Ascendex Global Trading - Coming Soon Page

A premium, responsive "Coming Soon" landing page designed for **Ascendex Global Trading**. This page features a countdown timer to deployment, an integrated email subscription form, and a built-in corporate Light/Dark mode toggle.

## 🛠️ Features
* **Corporate Palette:** Tailored precisely using the corporate Deep Navy Blue and Metallic Gold from `ASCENDEX-GLOBAL.png`.
* **Dynamic Theme Switcher:** Fully functional toggle matching Light Mode (default) and Dark Mode.
* **Live Countdown Timer:** Configured to dynamically count down to the platform's launch date.
* **Fully Responsive:** Designed using Tailwind CSS for a seamless presentation across mobile, tablet, and desktop screens.
